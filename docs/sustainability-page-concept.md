## Section 3: Our Practices

### Title: "How We Maintain the Balance"

**Visual Treatment:** Side-scrolling cards on mobile, grid on desktop

#### Practice 1: Selective Harvesting
**Icon:** Single tree among many
**Visual:** Before/after showing minimal forest impact
"We remove individual trees, not entire areas. The forest canopy remains, protecting soil and smaller trees."

#### Practice 2: Seasonal Timing
**Icon:** Calendar with seasons
**Visual:** Winter harvesting scene
"We harvest in winter when sap is down and ground is frozen, minimizing damage to forest floors and root systems."

#### Practice 3: Natural Regeneration
**Icon:** Seeds dispersing
**Visual:** Young trees growing among mature ones
"We allow natural seeding alongside our planting, maintaining genetic diversity and forest resilience."

#### Practice 4: Zero Waste Workshop
**Icon:** Circular arrow
**Visual:** Sawdust becoming forest mulch
"Wood shavings become animal bedding, then compost. Offcuts become smaller products. Bark returns to forest floor."

#### Practice 5: Local Focus
**Icon:** Map pin in forest
**Visual:** Map showing forest to workshop distance
"Our forests are within 50km of our workshop. Shorter transport means lower emissions and deeper connection to the land."

#### Practice 6: Chemical-Free Processing
**Icon:** Leaf checkmark
**Visual:** Natural finishing process
"We use heat treatment for pest control, natural oils for finishing. Nothing that wouldn't decompose safely in soil."

## Interactive Elements

### CSS-Only Animations

**Falling Leaves:**
```css
@keyframes falling-leaves {
  0% {
    transform: translateY(-100vh) rotate(0deg);
    opacity: 0;
  }
  10% {
    opacity: 1;
  }
  90% {
    opacity: 1;
  }
  100% {
    transform: translateY(100vh) rotate(360deg);
    opacity: 0;
  }
}

.leaf {
  animation: falling-leaves 15s linear infinite;
  animation-delay: calc(random() * 10s);
}
```

**Growth Animation:**
```css
.growth-stat {
  animation: grow 2s ease-out;
}

@keyframes grow {
  from {
    transform: scale(0);
    opacity: 0;
  }
  to {
    transform: scale(1);
    opacity: 1;
  }
}
```
