Submit extension

Submit your extension

Steps:
  1. Submit all required assets and info
  2. Extension is reviewed to ensure it meets quality criteria
  3. If approved the extension will be listed in the directory
  4. If rejected feedback will be provided
Theme requirements
  • Theme must have proper layout, spacing and styling
  • It must be aesthetically pleasing, outdated or lack of proper design is not accepted
  • It must include all essential templates (html files) to provide full functionality
  • Proper page builder support with sections/header/footer included in body tag, sections support with included screenshots
Plugin requirements
  • Plugin should not have telemetry or collect user data
  • Lite versions, upsells, adware, nagware or bloated extensions are not accepted
  • Plugin should use the same dashboard styling for configuration pages, no custom ui to confuse users

Extensions must be provided under an open source license.

  • For themes MIT or Apache 2.0 is recommended.
  • For plugins the recommendation is GPL.
