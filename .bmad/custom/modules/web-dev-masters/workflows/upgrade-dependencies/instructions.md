# Dependency Upgrade Protocol

<workflow>
<step n="1" goal="Audit Dependencies"><action>Check outdated packages</action><action>Review security vulnerabilities</action><action>Identify breaking changes</action></step>
<step n="2" goal="Prioritize Updates"><action>Security fixes first</action><action>Major versions carefully</action><action>Batch minor updates</action></step>
<step n="3" goal="Test Upgrades"><action>Update in dev environment</action><action>Run full test suite</action><action>Check type errors</action></step>
<step n="4" goal="Deploy Safely"><action>Deploy to staging</action><action>Monitor for issues</action><action>Keep rollback ready</action></step>
<step n="5" goal="Document Changes"><action>Update changelog</action><action>Note breaking changes</action></step>
</workflow>
