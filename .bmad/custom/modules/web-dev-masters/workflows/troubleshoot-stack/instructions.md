# Stack Troubleshooting Protocol

<workflow>
<step n="1" goal="Issue Identification"><ask>What's failing?</ask><ask>Error messages?</ask><ask>When does it fail?</ask></step>
<step n="2" goal="Isolate Layer"><action>Determine layer (frontend/backend/database)</action><action>Check logs</action></step>
<step n="3" goal="Systematic Debug"><action>Reproduce issue</action><action>Check recent changes</action><action>Verify configuration</action></step>
<step n="4" goal="Root Cause Analysis"><action>Identify root cause</action><action>Test fix</action></step>
<step n="5" goal="Prevention"><action>Add monitoring</action><action>Add tests</action><action>Document solution</action></step>
</workflow>
